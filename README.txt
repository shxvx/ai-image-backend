# AI Image Generator

### How to Run

1. Replace your Hugging Face token in `server.js`:
   ```
   const HUGGINGFACE_TOKEN = "hf_your_token_here";
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Start backend:
   ```
   node server.js
   ```

4. Serve `index.html` with Live Server, http-server, or just open in browser.

5. Go to:
   ```
   http://localhost:8080
   ```

6. Enter a prompt and generate your AI image!
